Documentation for Helfy

A project by Jugend Hackt

MySQL Database

Helfy requiers a database with two tables ("users" and "mitfahren"). Configure the passwords.php file to adjust Helfy to your database (see passwords.sample.php)

users has got the following collumns: `id`, `username`, `password`, `session`
mitfahren has got the following collumns: `id`, `type`, `mitfahrer_id`, `fahrer_id`, `start`, `ziel`, `description`, `timestamp`

Backend

You can access the backend using GET requests. Sending e.g. request=registrateUser&username=benutzername&password=passwort to it will create a new user.

registrateUser
requires: username, password

checkLoginData
requires: username, password

newSeccion
requires: username, password
returns: session_id

addRide
requires: username, session_id, type, start, ziel, description
returns: success/failed

nearbyRides
requires: username, session_id, lat, lon
returns: csv of results (fahrt_id, type, mitfahrer_id, fahrer_id, start, ziel, description, timestamp)

editRide
requires: fahrt_id, session_id, username, type, mitfahrer_id, fahrer_id, start, ziel, description, timestamp
returns: success/failed
